# COMANDOS DO GITBASH
## GIT CONFIG --LIST
### _Lista as configurções do git bash._

## GIT CONFIG --GLOBAL USER.NOME "NOME"
### _Configura o nome do usuário._

## GIT CONFIG --GLOBAL USER.EMAIL "E-MAIL"
### _Configura o e-mail do usuário._

# COMANDO NO TERMINAL(DESKTOP)
## GIT CLONE LINK_DO_REPOSITÓRIO
### _Abrir pasta na Área de Trabalho_


# COMANDOS GITHUB
## GIT STATUS
### _Diz se a alteração do arquivo esta salva ou não_

## GIT ADD NOME-DO-ARQUIVO
### _Salva o arquivo_ 

## GIT COMMIT -M "DESCRIÇÃO"
### _Salva a alteração_

## GIT PUSH 
### _"joga" a alteração no git hub_

## GIT PULL
### _"joga" a alteração do git hub na pasta do computador_
